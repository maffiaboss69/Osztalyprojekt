# osztályprojekt
11.E próba projekt

"Fork" (Másolás): Kattints a Fork gombra a projektemnél!
(Ezzel mindenki kap egy saját másolatot a saját fiókjába, ahol bátran kísérletezhet.)
"Commit" (Mentés): Hozzatok létre egy új fájlt a saját másolatotokban vendegek/sajat-nev.md néven, és írjatok bele pár sort magatokról.
Kattintsatok a Commit changes gombra. Ez olyan, mint egy mentés.

"Pull Request" (Egyesítés): Ez a legfontosabb rész. Küldjétek vissza a változtatásaitokat nekem a Pull Request (PR) gombbal.
